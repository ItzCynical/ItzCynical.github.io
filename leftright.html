· Left or Right ·

Project Overview

Left or Right was our group's project for VGDC from Fall 2016 - March 2017. The idea of the game was an interactive story driven game that centered the interaction through phones. We set up a gmail account and Google voice number that allows players to message it to start the game/receive texts back that would tell the story. We had several command words that would allow the players to inspect certain objects described in the text messages and multiple rooms the player could traverse.

Skills Applied

  • Javascript
  • Google Voice
  • Google API

Personal Contributions

  • Utilized google scripts to pull text messages received via Google voice and map them to a spreasheet that served as a database of player progress in the story.
  • Divided the rooms of the game into separate functions each containing responses to key words found in player's messages.
